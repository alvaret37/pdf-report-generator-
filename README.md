# 📄 PDF Report Generator

Gere relatórios em PDF com tabelas e gráficos a partir de dados CSV.

## 📦 Estrutura
```
pdf_report_generator/
├── app/
│   └── main.py
├── data/
│   └── sample.csv
├── reports/
├── tests/
├── .gitignore
├── requirements.txt
└── README.md
```

## ▶️ Como rodar

```bash
python app/main.py
```

## ✅ Resultado esperado

Um relatório PDF com gráfico de barras será gerado na pasta `reports`.
